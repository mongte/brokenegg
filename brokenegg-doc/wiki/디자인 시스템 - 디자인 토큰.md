# 디자인 토큰

실제 정의 위치: `apps/brokenegg-app/src/app/global.css`의 `:root`. **신규 페이지에서 색/반경/서체를 직접 하드코딩하지 말고 이 변수를 쓴다.**

> 상위: [[디자인 시스템 - 개요]] · 같이: [[디자인 시스템 - 컴포넌트]]

## 색 (Color)
| 토큰 | 값 | 용도 |
|---|---|---|
| `--bg` | `#ffffff` | 캔버스/밝은 면 배경 |
| `--frame-bg` | `#f2f2f2` | 페이지 최외곽(캔버스 바깥) 프레임 |
| `--text-main` | `#000000` | 본문/제목, 1차 버튼 배경, 검정 섹션 배경 |
| `--text-muted` | `#666666` | 보조 텍스트, 메타, 푸터 |
| `--accent-bg` | `#efefef` | 배지 기본 배경, hover 면 |
| `--border` | `#e0e0e0` | 섹션 구분선(1px), 보조 버튼 테두리 |

추가 회색: 카드 hover `#fafafa`, 검정 섹션 위 보더 `rgba(255,255,255,0.1)`. **컬러 액센트는 도입하지 않는다.**

## 간격 (Spacing) — 관찰된 리듬
- 페이지 바깥 여백: `body { padding: 20px }`
- 헤더: `32px 40px`
- 일반 섹션 좌우 패딩: `40px` (`hero`, `section-header`, `testimonials`, `footer`)
- 카드 내부: `40px`(problem-card), `20px`(service overlay)
- 큰 세로 여백: testimonials `80px`, final-cta `120px`
- 가로 정렬 좌우 표준값은 **40px**. 새 섹션도 좌우 40px를 기본으로.

## 모션
- 버튼/면 전환: `all 0.2s ease`, 1차 버튼 hover `translateY(-1px)`
- 미디어 호버: `0.6s cubic-bezier(0.23,1,0.32,1)` (그레이스케일→컬러 + scale 1.05)
- 슬라이드 패널: `height 0.45s cubic-bezier(0.4,0,0.2,1)`

## 반응형 브레이크포인트
- **단일 브레이크포인트 `max-width: 1024px`**: 3·4열 그리드 → 1열, 우측 보더 → 하단 보더로 전환, 서비스 카드 비율 `4/5` → `16/9`.
- 그 이상에서는 `.canvas` max-width 1440px 안에서 유동.

## 장식
- `.grid-bg`: 화면 고정, 100px 격자, `rgba(0,0,0,0.02)` 1px 선, `z-index:-1`. 모든 페이지 공통 배경.
